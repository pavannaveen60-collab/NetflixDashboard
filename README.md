# 🎬 Netflix Dashboard

An interactive Netflix Dashboard built using **R** and **Shiny** to analyze and visualize the Netflix Titles dataset.

## 📌 Features

- 📊 Interactive dashboard
- 🎭 Genre-wise analysis
- 🌍 Country-wise content distribution
- 📅 Content added over the years
- 🎬 Movies vs TV Shows comparison
- ⭐ Clean and responsive interface

## 🛠️ Tech Stack

- R
- Shiny
- ggplot2
- dplyr
- DT

## 📂 Dataset

Dataset used:
- `netflix_titles.csv`

Source:
https://www.kaggle.com/datasets/shivamb/netflix-shows

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/pavannaveen60-collab/NetflixDashboard.git
```

### Open the project

Open `NetflixDashboard.Rproj` in RStudio.

### Install dependencies

```R
install.packages(c("shiny","ggplot2","dplyr","DT"))
```

### Run the application

```R
shiny::runApp()
```

or simply click **Run App** in RStudio.
